# Enterprise Integration Pattern Shapes for Gliffy

<a href="https://codeclou.github.io/enterprise-integration-pattern-shapes-for-gliffy/"><img src="https://codeclou.github.io/enterprise-integration-pattern-shapes-for-gliffy/images/github-product-logo-eip-shapes-gliffy.png" width="100%"/></a>

This project provides [SVG](https://de.wikipedia.org/wiki/Scalable_Vector_Graphics)-shapes for the [Enterprise Integration Patterns](https://www.enterpriseintegrationpatterns.com/toc.html) that are intended to be used with [Gliffy](https://www.gliffy.com).

See the [project page](https://codeclou.github.io/enterprise-integration-pattern-shapes-for-gliffy/) for full documentation and **[usage guide](https://codeclou.github.io/enterprise-integration-pattern-shapes-for-gliffy/#usage-guide)**.

Here is a picture of some shapes so that you get an idea:
